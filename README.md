# Nguyên Lý Hệ Điều Hành
1. Lý Thuyết
  - Tổng quan
  - Cấu Trúc
  - Quản lý tiến trình
  - Định thời CPU
2. Thực Hành
  - Lý thuyết thực hành
  - Bài tập thực hành
3. Báo Cáo(tuần 9)
  - Bài tập nhóm (giả lập)
4. Slide bài giảng (TS Trần Công Án && TS Trần Cao Đệ)
5. Đánh giá
  - Bài tập 10%
  - Lý thuyết 60%
  - Thực hành 30%
6. Lịch học
  - Học 3 buổi lý thuyết 1 buổi thực Hành
  - Tuần 9 báo Cáo
  - Tuần 16 thi kết thúc học phần
7. Chia nhóm (Nhóm 1)
  - Nhóm [Zalo](https://zaloapp.com/g/dmsvnl259)
8. Download tài liệu
